# ZZZ Achievement Tracker (React)

Сборка на Vite + React. База под GitHub Pages: `/achievments/`.

## Запуск
```bash
npm i
npm run dev
```

## Сборка
```bash
npm run build
npm run preview
```

## GitHub Pages
В `vite.config.js` уже стоит:
- `base: '/achievments/'`

Дальше можно деплоить содержимое папки `dist/` в ветку `gh-pages` или в `docs/` (как тебе удобнее).
